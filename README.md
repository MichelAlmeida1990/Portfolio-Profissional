## ⚙️ Personalização

<div align="center">

![Customization](https://media.giphy.com/media/LnQjpWaON8nhr21vNW/giphy.gif)

</div>

### 🎨 Como Personalizar

1. **📝 Informações Pessoais**
   - Substitua as informações no arquivo `index.html`
   - Atualize a foto de perfil em `image/foto perfil.png`

2. **💼 Projetos**
   - Modifique os projetos na seção "Meus Projetos"
   - Atualize as imagens na pasta `image/projects/`

3. **🏆 Certificações**
   - Atualize as certificações na seção "Certificados"
   - Adicione novos certificados ao carrossel

4. **🎨 Estilos e Cores**
   - Personalize as cores em `styles.css`
   - Use as variáveis CSS no início do arquivo para facilitar a customização

## 🧩 Componentes Principais

### ✨ Efeito de Partículas
O efeito de partículas na seção hero é criado usando a biblioteca **particles.js**. A configuração pode ser ajustada no arquivo `js/particles/particles-config.js`.

```javascript
// Exemplo de configuração
particlesJS("particles-js", {
  particles: {
    number: { value: 80 },
    color: { value: "#ffffff" },
    // ... outras configurações
  }
});
```

### 🖱️ Cursor Personalizado
O cursor personalizado com efeito de rastro é implementado com JavaScript puro. A lógica está no arquivo `js/particles/cursor.js` e os estilos em `css/cursor.css`.

### 🎠 Carrossel de Certificados
Para adicionar um novo certificado ao carrossel:

```html
<div class="certificate-slide">
  <img src="image/certificado.jpg" alt="Certificado">
  <h3>Nome do Certificado</h3>
  <p>Instituição</p>
</div>
```

### 🎭 Animações AOS
As animações de entrada são controladas pela biblioteca **AOS (Animate On Scroll)**. Ajuste os efeitos modificando os atributos:

```html
<div data-aos="fade-up" data-aos-delay="200" data-aos-duration="800">
  <!-- Conteúdo animado -->
</div>
```
